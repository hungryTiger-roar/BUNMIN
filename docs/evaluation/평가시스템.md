# AI 모델 평가 시스템

ASR, NMT, TTS, OCR 모델의 품질(WER, BLEU, CER)과 속도(RTF, 지연시간)를 측정합니다.

---

## 사전 준비

conda 환경이 설치되어 있어야 합니다.

```bash
cd backend
conda activate aunion
```

---

## 데이터셋 다운로드

최초 1회 실행합니다.

```bash
python evaluation/download_datasets.py
```

다운로드 위치: `evaluation/datasets/`

---

## 평가 실행

```bash
python evaluation/run_eval.py
```

실행 시 인터랙티브 메뉴가 표시됩니다:

```
평가할 모델을 선택하세요:
  1. ASR  - 음성인식 (WER)
  2. NMT  - 번역 (BLEU)
  3. TTS  - 음성합성 (CER, RTF)
  4. OCR  - 문자인식 (CER)
  5. Pipeline - 실시간 파이프라인 통합
  6. OCR+NMT  - 슬라이드 번역 파이프라인
  a. 전체 평가
  q. 종료
```

---

## 특정 모델만 평가

```bash
python evaluation/run_eval.py --model asr
python evaluation/run_eval.py --model nmt
python evaluation/run_eval.py --model tts
python evaluation/run_eval.py --model ocr
```

---

## 이전 결과와 비교

```bash
python evaluation/run_eval.py --compare
```

---

## 결과 확인

- 최신 결과: `evaluation/results/latest.json`
- 이력 결과: `evaluation/results/history/eval_YYYYMMDD_HHMMSS.json`

---

## 평가 지표

| 모델 | 지표 | 설명 |
|------|------|------|
| ASR | WER | 단어 오류율 (낮을수록 좋음) |
| NMT | BLEU | 번역 품질 점수 (높을수록 좋음) |
| TTS | CER | 문자 오류율 (낮을수록 좋음) |
| OCR | CER | 문자 오류율 (낮을수록 좋음) |
| 전체 | RTF | 실시간 처리 비율 (1.0 이하 = 실시간 가능) |

---

## 디렉토리 구조

```
evaluation/
├── run_eval.py              # 평가 진입점 (인터랙티브 메뉴)
├── download_datasets.py     # 데이터셋 다운로드
├── generate_samples.py      # OCR 샘플 생성
├── evaluators/
│   ├── asr.py               # ASR 평가
│   ├── nmt.py               # NMT 평가
│   ├── tts.py               # TTS 평가
│   ├── ocr.py               # OCR 평가
│   └── pipeline.py          # 통합 파이프라인 평가
├── metrics/
│   ├── wer.py               # WER 계산
│   ├── bleu.py              # BLEU 계산
│   ├── cer.py               # CER 계산
│   └── speed.py             # 속도 측정
├── datasets/                # 평가 데이터셋 (다운로드 후 생성)
└── results/                 # 평가 결과
    ├── latest.json
    └── history/
```
