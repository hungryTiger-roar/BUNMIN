# AI 모델 평가 시스템

ASR, NMT, TTS, OCR 모델의 품질(WER, BLEU, CER)과 속도(RTF, 지연시간)를 측정합니다.

---

## 사전 준비

conda 환경이 설치되어 있어야 합니다.

```bash
conda activate aunion
```

> **주의**: 반드시 conda aunion 환경에서 실행해야 합니다.
> `python -m evaluation.run_eval`은 잘못된 Python 환경 감지 시 자동으로 conda aunion으로 전환합니다.

---

## 데이터셋 다운로드

최초 1회 실행합니다.

```bash
python evaluation/download_datasets.py
```

다운로드 위치: `evaluation/datasets/`

---

## 평가 실행

프로젝트 루트에서 실행합니다.

```bash
python -m evaluation.run_eval
```

실행 시 인터랙티브 메뉴가 표시됩니다:

```
실행 환경 선택:
  1. CPU
  2. GPU

평가할 모델을 선택하세요:
  1. ASR      (음성인식)
  2. NMT      (번역)
  3. TTS      (음성합성)
  4. OCR      (문자인식)
  5. Pipeline (ASR→NMT→TTS 전체)
  6. OCR+NMT  (슬라이드 번역)
  7. 전체 평가
  8. 이전 결과와 비교
```

TTS 평가 시 생성된 오디오 파일이 자동 저장됩니다:
`evaluation/datasets/tts_samples/output/sample_NN.wav`

---

## 특정 모델만 평가

```bash
python -m evaluation.run_eval --model asr
python -m evaluation.run_eval --model nmt
python -m evaluation.run_eval --model tts
python -m evaluation.run_eval --model ocr
```

---

## 이전 결과와 비교

```bash
python -m evaluation.run_eval --compare
```

- 최신 결과: `evaluation/results/latest.json`
- 이력 결과: `evaluation/results/history/eval_YYYYMMDD_HHMMSS.json`

## 평가 지표

| 모델 | 지표 | 설명 |
|------|------|------|
| ASR | WER | 단어 오류율 (낮을수록 좋음) |
| NMT | BLEU | 번역 품질 점수 (높을수록 좋음) |
| TTS | RTF | 실시간 처리 비율 (1.0 이하 = 실시간 가능) |
| OCR | CER | 문자 오류율 (낮을수록 좋음) |
| 전체 | RTF | 실시간 처리 비율 (1.0 이하 = 실시간 가능) |

## 디렉토리 구조

```
backend/evaluation/
├── run_eval.py              # 평가 진입점 (인터랙티브 메뉴)

│   ├── asr.py               # ASR 평가
│   ├── nmt.py               # NMT 평가
│   ├── tts.py               # TTS 평가
│   ├── ocr.py               # OCR 평가
│   └── pipeline.py          # 통합 파이프라인 평가
├── metrics/
│   ├── wer.py               # WER 계산
│   ├── bleu.py              # BLEU 계산
│   ├── cer.py               # CER 계산
│   └── speed.py             # 속도 측정
├── datasets/                # 평가 데이터셋 (다운로드 후 생성)

    ├── latest.json

```

## OCR 강의 슬라이드 데이터셋

RapidOCR의 대학교 강의 슬라이드 도메인 성능을 평가하기 위한 별도 데이터셋입니다.

### 데이터셋 구성

MIT OpenCourseWare 공개 강의 PDF를 기반으로 다음 분야를 포함합니다:

| 분야 | 과목 |
|------|------|
| 컴퓨터공학 | Introduction to CS (6.0001) |
| 수학 | Linear Algebra, Probability |
| 자연과학 | Physical Chemistry, Relativity |
| 경제학 | International Trade, Behavioral Economics |
| 인문/심리 | Introduction to Psychology |
| AI/알고리즘 | Machine Learning, Algorithms |

한국어 강의는 [KOCW](https://www.kocw.net)에서 직접 다운로드 후 `evaluation/pdfs/` 폴더에 추가합니다.

### 실행 순서

```bash
cd backend/evaluation

# 1. MIT OCW 강의 PDF 다운로드
python download_pdfs.py

# 2. Ground Truth + 슬라이드 이미지 생성
python build_dataset.py

# 3. OCR 평가 실행
pip install editdistance
python evaluate_ocr.py
```

결과: `evaluation/eval_result.json` — 슬라이드별 CER/WER 및 평균값 출력
