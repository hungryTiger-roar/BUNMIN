# 백엔드 설계

## 개요

FastAPI 기반 AI 백엔드 서버.
웹 개발 환경에서는 독립 실행되며, Electron 배포 시 PyInstaller로 단일 exe에 번들링됩니다.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Web Framework | FastAPI + uvicorn |
| 실시간 통신 | WebSocket |
| AI - ASR | faster-whisper / CTranslate2 (ghost613/faster-whisper-large-v3-turbo-korean) |
| AI - NMT (실시간) | transformers / MarianMT (Helsinki-NLP/opus-mt-ko-en) |
| AI - 슬라이드 번역 | Qwen3-VL-8B + LoRA (VLM 기반, EasyOCR + 번역 통합) |
| AI - TTS | Piper TTS (en_US-lessac-medium.onnx) |
| AI - OCR | RapidOCR (ONNX 기반, CPU 최적화) |
| PDF 처리 | PyMuPDF |
| 빌드 | PyInstaller |

---

## API 명세

### 슬라이드

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/slides/upload` | PDF 업로드 → 백그라운드 VLM 번역 시작 |
| GET | `/slides/status/{id}` | 처리 상태 확인 (pending/processing/completed/failed) |
| GET | `/slides/pages/{id}` | 번역된 페이지 전체 데이터 |
| GET | `/slides/image/{id}/{page}` | 슬라이드 페이지 이미지 반환 (`?translated=true`로 번역본) |
| GET | `/slides/download/{id}` | PDF 다운로드 (`?type=original` 또는 `?type=translated`) |

### 자막

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/transcripts` | 저장된 강의 세션 목록 (started_at 내림차순) |
| GET | `/transcripts/{session_id}` | 세션 자막 전체 JSON 조회 (종료된 세션만) |
| GET | `/transcripts/{session_id}/download` | 자막 다운로드 (`?format=json\|srt\|txt`) |
| DELETE | `/transcripts/{session_id}` | 자막 삭제 |

### 네트워크

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/network/info` | 강의자 PC의 LAN IP + 수강자 초대 URL 반환 |

### 모드 전환

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/mode/current` | 현재 모드 + 로드된 모델 목록 확인 |
| POST | `/api/mode/slide` | 슬라이드 번역 모드로 전환 (실시간 모델 언로드) |
| POST | `/api/mode/realtime` | 실시간 번역 모드로 전환 (VLM 언로드 → ASR/NMT/TTS/OCR 로드) |

### 번역 (직접 API)

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/translate/slide` | 이미지 파일 업로드 → 번역된 이미지 반환 |
| POST | `/translate/slide/json` | 이미지 파일 업로드 → JSON 응답 (regions/bbox 포함) |

### WebSocket

| 경로 | 설명 |
|------|------|
| `/ws/pipeline` | 강의자/수강자 통합 실시간 처리 |

### 헬스체크

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET / HEAD | `/health` | 모델 로딩 상태 및 진행률 확인 |
| GET(thread) | `port 18765` | GIL 무관 전용 health 서버 |

## WebSocket 메시지 형식

### 클라이언트 → 서버

```json
// 역할 등록 (첫 메시지, 필수)
{ "type": "register", "role": "lecturer" }
{ "type": "register", "role": "student" }

// 음성 전송 (강의자)
{ "type": "audio", "audio": "<base64 WAV>", "sample_rate": 16000, "sentAt": 1710000000000 }

// 화면 공유 프레임 (강의자)
{ "type": "screen", "data": "<base64 JPEG>" }

// 슬라이드 선택 (강의자)
{ "type": "slide_select", "slide_id": "abc123" }

// 페이지 변경 (강의자)
{ "type": "page_change", "slide_id": "abc123", "page": 3 }

// 강의 제어 (강의자)
{ "type": "lecture_start", "slide_id": "abc123", "mode": "slide" }
{ "type": "lecture_pause" }
{ "type": "lecture_resume" }
{ "type": "lecture_end" }

// 발표 모드 변경 (강의자)
{ "type": "presentation_mode", "mode": "slide" }

// heartbeat 응답 (강의자/수강자 공통)
{ "type": "pong" }
```

### 서버 → 클라이언트

```json
// 역할 등록 확인 (강의자/수강자 공통)
{ "type": "registered", "role": "lecturer" }

// 강의 세션 시작 확인 (강의자에게만)
{ "type": "session_started", "session_id": "a1b2c3d4" }

// 번역 결과 (수강자)
{
  "type": "transcription",
  "original": "한국어 텍스트",
  "translated": "English text",
  "audio": "<base64 WAV>",
  "sentAt": 1710000000000
}

// 화면 공유 프레임 (수강자)
{ "type": "screen", "image": "<base64 JPEG>", "slide_id": "abc123", "page": 3 }

// 슬라이드 오버레이 (수강자, 슬라이드 모드 시 사전 처리된 VLM 결과)
{
  "type": "overlay",
  "items": [
    { "original": "한국어", "translated": "English", "bbox": [x, y, w, h], "confidence": 0.9 }
  ]
}

// 강의 제어 동기화 (수강자)
{ "type": "lecture_start", "slide_id": "abc123", "session_id": "a1b2c3d4" }
{ "type": "lecture_pause" }
{ "type": "lecture_resume" }
{ "type": "lecture_end", "session_id": "a1b2c3d4" }
{ "type": "page_change", "slide_id": "abc123", "page": 3 }
{ "type": "slide_select", "slide_id": "abc123" }
{ "type": "presentation_mode", "mode": "screen" }

// 수강자 접속 수 (수강자 전체, 접속/해제 시 갱신)
{ "type": "student_count", "count": 5 }

// heartbeat (서버 → 클라이언트, 20초 주기)
{ "type": "ping" }
```

> **audio 필드**: TTS 실패 시 `null`로 전송 — 수강자 측은 자막만 표시

## AI 파이프라인

### 음성 번역 (실시간)

```
마이크 입력 → ASR → NMT → TTS → 번역 음성 + 자막 출력
             (한→텍) (한→영) (영→음성)
```

TTS 합성 실패 시에도 자막은 정상 전송됩니다 (`audio: null`).

### 슬라이드 번역

```
[강의 전 — PDF 업로드 시]
PDF 업로드
  → 페이지별 PNG 이미지 변환 (PyMuPDF)
  → EasyOCR (텍스트 영역 감지)
  → Qwen3-VL(LoRA) (한→영 번역, 번역 이미지 합성)
  → 번역 이미지 저장 + overlay_items 생성 (원문/번역/bbox)

[강의 중 — 화면공유 프레임 수신 시]
화면공유 프레임 → 수강자 전달
                 ↘ 슬라이드 모드 시: 사전 처리된 overlay_items 전송
```

실시간 OCR+번역 없음 — 강의 중 슬라이드 처리는 모두 사전 저장된 데이터를 반환합니다.

## 모델 로딩 순서

앱 시작 시 2단계로 로딩 (로딩 화면에 진행률 표시):

**1단계: 병렬 다운로드** — 미캐시 모델만 병렬로 HuggingFace 다운로드

**2단계: 순차 메모리 로딩**
1. ASR (ghost613/faster-whisper-large-v3-turbo-korean) — 1/4
2. NMT-ASR (Helsinki-NLP/opus-mt-ko-en) — 2/4
3. TTS (Piper TTS) — 3/4
4. OCR (RapidOCR) — 4/4

슬라이드 번역(Qwen3-VL)은 PDF 업로드 시점에 온디맨드로 로드됩니다.

`SKIP_STARTUP_MODELS=true` 설정 시 ASR/NMT/TTS/OCR을 건너뛰고 즉시 준비 상태로 전환됩니다 (슬라이드 번역 전용 모드).

로딩 상태는 `/health` 엔드포인트 및 `__AUNION_STATUS__` stdout 신호로 Electron에 전달됩니다.
port 18765 전용 스레드 서버는 GIL에 무관하게 항상 응답합니다.

## 저장 방식

| 항목 | 저장 방식 |
|------|-----------|
| 슬라이드 데이터 | 메모리 (Python dict) + 파일시스템 |
| 슬라이드 PDF | `uploads/slides/{slide_id}.pdf` |
| 슬라이드 이미지 (원본) | `uploads/images/{slide_id}_{page}.png` |
| 슬라이드 이미지 (번역본) | `uploads/translated/{slide_id}_{page}.png` |
| 번역 PDF | `uploads/translated/{slide_id}_translated.pdf` |
| 강의 자막 | `uploads/transcripts/{session_id}.jsonl / .json / .meta.json` |
| 모델 캐시 | 개발: `backend/cache/huggingface` / 배포: `%LOCALAPPDATA%\Aunion AI\cache\huggingface` |
| 에러 로그 | `%LOCALAPPDATA%\Aunion AI\error_log.txt` |

> `uploads/` 경로는 서버 실행 위치 기준 상대 경로입니다. `npm run dev`로 실행 시 프로젝트 루트에 생성됩니다.
