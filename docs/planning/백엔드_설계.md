# 백엔드 설계

## 개요

Electron 데스크탑 앱 내장 FastAPI 서버.
PyInstaller로 단일 exe에 번들링되어 배포됩니다.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Web Framework | FastAPI + uvicorn |
| 실시간 통신 | WebSocket |
| AI - ASR | faster-whisper (ghost613/faster-whisper-large-v3-turbo-korean) |
| AI - NMT | transformers (Helsinki-NLP/opus-mt-ko-en) |
| AI - TTS | optimum + onnxruntime (onnx-community/Supertonic-TTS-2-ONNX) |
| AI - OCR | RapidOCR |
| PDF 처리 | PyMuPDF |
| 빌드 | PyInstaller |

---

## API 명세

### 슬라이드

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/slides/upload` | PDF 업로드 → 백그라운드 OCR+번역 |
| GET | `/slides/status/{id}` | 처리 상태 확인 |
| GET | `/slides/pages/{id}` | 번역된 페이지 데이터 |
| GET | `/slides/download/{id}` | PDF 다운로드 |

### WebSocket

| 경로 | 설명 |
|------|------|
| `/ws/pipeline` | 강의자/수강자 통합 실시간 처리 |

### 헬스체크

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/health` | 모델 로딩 상태 및 진행률 확인 |

## WebSocket 메시지 형식

### 클라이언트 → 서버 (강의자)

```json
// 역할 등록
{ "type": "register", "role": "lecturer" }

// 음성 전송
{ "type": "audio", "audio": "<base64 PCM>", "sample_rate": 16000 }

// 화면 공유 프레임
{ "type": "screen", "data": "<base64 JPEG>" }

// 슬라이드 선택
{ "type": "slide_select", "slide_id": "abc123" }

// 페이지 변경
{ "type": "page_change", "slide_id": "abc123", "page": 3 }

// 강의 제어
{ "type": "lecture_start", "slide_id": "abc123", "mode": "slide" }
{ "type": "lecture_pause" }
{ "type": "lecture_resume" }
{ "type": "lecture_end", "slide_id": "abc123" }

// 발표 모드 변경
{ "type": "presentation_mode", "mode": "slide" }
```

### 서버 → 클라이언트 (수강자)

```json
// 번역 결과 (자막 + 음성)
{
  "type": "transcription",
  "original": "한국어 텍스트",
  "translated": "English text",
  "audio": "<base64 WAV>"
}

// 화면 공유 프레임
{ "type": "screen", "image": "<base64 JPEG>" }

// 강의 제어 동기화
{ "type": "lecture_start", "slide_id": "abc123" }
{ "type": "lecture_pause" }
{ "type": "lecture_resume" }
{ "type": "lecture_end" }
{ "type": "page_change", "page": 3 }
{ "type": "presentation_mode", "mode": "screen" }
```

## 디렉토리 구조

```
backend/
├── app/
│   ├── config.py            # 모델 설정, 디바이스 결정, 경로 설정
│   ├── main.py              # FastAPI 앱, 모델 로딩, 헬스체크
│   ├── routers/
│   │   ├── slides.py        # PDF 업로드/처리/다운로드
│   │   └── ws.py            # WebSocket 실시간 파이프라인
│   └── services/
│       ├── asr_service.py   # 음성인식 (faster-whisper)
│       ├── nmt_service.py   # 번역 (Helsinki-NLP)
│       ├── tts_service.py   # 음성합성 (Supertonic-TTS-2 ONNX)
│       └── ocr_service.py   # 문자인식 (RapidOCR)
├── aunion.spec              # PyInstaller 빌드 설정
├── run.py                   # 진입점 (에러 로그, uvicorn 기동)
└── requirements.txt
```

## AI 파이프라인

### 음성 번역 (실시간)

```
마이크 입력 → ASR → NMT → TTS → 번역 음성 + 자막 출력
             (한→텍) (한→영) (영→음성)
```

### 슬라이드 번역

```
[강의 전]
PDF 업로드 → 페이지별 이미지 변환 → OCR → NMT → 저장

[강의 중]
슬라이드 페이지 번호 → 저장된 번역 반환 (실시간 OCR 없음)
```

## 모델 로딩 순서

앱 시작 시 순차적으로 로딩 (로딩 화면에 진행률 표시):

1. ASR (faster-whisper) — 1/4
2. NMT (Helsinki-NLP) — 2/4
3. TTS (Supertonic-2 ONNX) — 3/4
4. OCR (RapidOCR) — 4/4

로딩 상태는 `/health` 엔드포인트 및 `__AUNION_STATUS__` stdout 신호로 Electron에 전달됩니다.

## 저장 방식

| 항목 | 저장 방식 |
|------|-----------|
| 슬라이드 데이터 | 메모리 (Python dict) + 파일시스템 |
| 모델 캐시 | `%LOCALAPPDATA%\Aunion AI\cache\huggingface` |
| 슬라이드 파일 | `%LOCALAPPDATA%\Aunion AI\slides` |
| 에러 로그 | `%LOCALAPPDATA%\Aunion AI\error_log.txt` |
