# 커서 공유 기능 설계

강의자의 마우스 포인터 위치를 실시간으로 수강자 화면에 표시하는 기능

## 개요

- 강의자가 슬라이드 위에서 마우스를 움직이면 수강자 화면에도 동일한 위치에 스팟라이트 커서가 표시됨
- 화면 크기/비율이 달라도 정확한 위치에 표시되도록 상대 좌표(0~1) 사용
- 수강자 화면 깜빡임 방지를 위해 ref 기반 DOM 직접 조작 방식 적용

## 좌표 계산 로직

### object-fit: contain 고려

슬라이드 이미지는 `object-fit: contain`으로 렌더링되어 letterbox/pillarbox 여백이 생길 수 있음.
실제 이미지 렌더링 영역을 계산하여 정확한 좌표 변환 수행.

```typescript
// 이미지 실제 렌더링 영역 계산
const img = container.querySelector('img')
const containerRect = container.getBoundingClientRect()

let imgOffsetX = 0, imgOffsetY = 0
let imgWidth = containerRect.width, imgHeight = containerRect.height

if (img && img.naturalWidth && img.naturalHeight) {
  const imgRatio = img.naturalWidth / img.naturalHeight
  const containerRatio = containerRect.width / containerRect.height

  if (imgRatio > containerRatio) {
    // 이미지가 더 넓음 → 좌우 맞춤, 위아래 여백
    imgWidth = containerRect.width
    imgHeight = containerRect.width / imgRatio
  } else {
    // 이미지가 더 좁음 → 위아래 맞춤, 좌우 여백
    imgHeight = containerRect.height
    imgWidth = containerRect.height * imgRatio
  }
  imgOffsetX = (containerRect.width - imgWidth) / 2
  imgOffsetY = (containerRect.height - imgHeight) / 2
}
```

### 강의자 → 상대좌표 변환

```typescript
const imgLeft = containerRect.left + imgOffsetX
const imgTop = containerRect.top + imgOffsetY
const relX = (e.clientX - imgLeft) / imgWidth  // 0~1
const relY = (e.clientY - imgTop) / imgHeight  // 0~1
```

### 수강자 → 픽셀 좌표 변환

```typescript
const px = imgOffsetX + relX * imgWidth
const py = imgOffsetY + relY * imgHeight
el.style.transform = `translate3d(${px}px, ${py}px, 0) translate(-50%, -50%)`
```

## 성능 최적화

### 1. 전송 빈도 제한 (20fps)

```typescript
const SEND_INTERVAL = 50 // ms
let lastSendTime = 0

const tick = () => {
  const now = Date.now()
  if (now - lastSendTime >= SEND_INTERVAL) {
    lastSendTime = now
    send({ type: 'cursor', x, y, visible, color })
  }
  rafId = requestAnimationFrame(tick)
}
```

### 2. ref 기반 DOM 직접 조작

React 상태 대신 ref를 사용하여 리렌더링 방지:

```typescript
export function useCursorOverlay(containerRef) {
  const spotlightRef = useRef<HTMLDivElement>(null)

  const onCursor = useCallback((cursor) => {
    const el = spotlightRef.current
    // React setState 없이 DOM 직접 조작
    el.style.transform = `translate3d(${px}px, ${py}px, 0)`
    el.style.opacity = cursor.visible ? '1' : '0'
  }, [])

  return { spotlightRef, onCursor }
}
```

### 3. GPU 가속 트랜스폼

```css
transform: translate3d(px, py, 0);  /* GPU 가속 */
will-change: transform, opacity;
```

## UI/UX

### 스팟라이트 스타일

- 크기: 40px 원형
- 테두리: 2px solid (선택 색상)
- 배경: radial-gradient (중심에서 바깥으로 투명해지는 그라데이션)
- 그림자: 내부/외부 glow 효과

### 색상 프리셋

```typescript
const SPOTLIGHT_PRESETS = [
  '#60A5FA', // sky blue (기본)
  '#F472B6', // pink
  '#FBBF24', // amber
  '#34D399', // emerald
  '#A78BFA', // purple
  '#EF4444', // red
]
```

### 여백 처리

- 마우스가 letterbox/pillarbox 여백에 있을 때 `visible: false` 전송
- 수강자 화면에서 커서가 부드럽게 사라짐 (opacity transition)
