# 프론트 설계 (MVP)

## 개요

화면 공유 + 음성 번역 핵심 기능만 구현.

- 강의자/수강자 얼굴(화상) 불필요
- 인증, 강의실 관리 등은 MVP 이후

---

## 플랫폼

| 플랫폼 | 기술 | 대상 |
|--------|------|------|
| 데스크톱 | Electron + React | 강의자 + 수강자 |
| 웹 | React | 수강자 |
| 모바일 | Jetpack Compose | 수강자 (MVP 이후) |

## 화면 구성

### 1. 메인 (역할 선택)

```
┌─────────────────────────────┐
│                             │
│     [ 강의자로 시작 ]        │
│                             │
│     [ 수강자로 참여 ]        │
│                             │
└─────────────────────────────┘
```

### 2. 강의자 화면

```
┌─────────────────────────────────────┐
│  [PDF 업로드]  [화면 공유]  [종료]   │
├─────────────────────────────────────┤
│                                     │
│       화면 공유 미리보기             │
│       (내가 공유 중인 화면)          │
│                                     │
├─────────────────────────────────────┤
│  실시간 자막                        │
│  [한] 오늘 수업 내용은...            │
│  [EN] Today's lesson is...          │
├─────────────────────────────────────┤
│  🎤 마이크: ON    📄 슬라이드: 준비됨 │
└─────────────────────────────────────┘
```

**기능:**
- PDF 업로드 (사전 처리)
- 화면 공유 시작/중지
- 마이크 ON/OFF
- 실시간 자막 미리보기

### 3. 수강자 화면

```
┌─────────────────────────────────────┐
│  [원본]  [번역 ✓]                   │
├─────────────────────────────────────┤
│                                     │
│    강의 화면 (번역 오버레이)         │
│                                     │
│    ┌─────────────────────────┐     │
│    │ Machine Learning Basics │     │
│    └─────────────────────────┘     │
│                                     │
├─────────────────────────────────────┤
│  📝 자막: Today we will learn...    │
├─────────────────────────────────────┤
│  🔊 음성: ON    📝 자막: ON          │
└─────────────────────────────────────┘
```

**기능:**
- 강의 화면 수신
- 원본/번역 토글
- 번역 음성 재생 ON/OFF
- 자막 표시 ON/OFF

## 핵심 기능 구현

### 1. 마이크 캡처 (강의자)

```typescript
// hooks/useAudioCapture.ts

const useAudioCapture = (onData: (chunk: Blob) => void) => {
  const [isCapturing, setIsCapturing] = useState(false)

  const start = async () => {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true })
    const recorder = new MediaRecorder(stream)

    recorder.ondataavailable = (e) => {
      onData(e.data)  // WebSocket으로 전송
    }

    recorder.start(500)  // 500ms 간격
    setIsCapturing(true)
  }

  return { isCapturing, start, stop }
}
```

### 2. 화면 캡처 (강의자)

```typescript
// hooks/useScreenCapture.ts

const useScreenCapture = (onCapture: (image: string) => void) => {
  const [isSharing, setIsSharing] = useState(false)

  const start = async () => {
    const stream = await navigator.mediaDevices.getDisplayMedia({ video: true })
    const video = document.createElement('video')
    video.srcObject = stream

    // 1초 간격으로 캡처
    setInterval(() => {
      const canvas = document.createElement('canvas')
      canvas.getContext('2d').drawImage(video, 0, 0)
      const image = canvas.toDataURL('image/jpeg', 0.7)
      onCapture(image)  // WebSocket으로 전송
    }, 1000)

    setIsSharing(true)
  }

  return { isSharing, start, stop }
}
```

### 3. WebSocket 통신

```typescript
// hooks/useWebSocket.ts

const useWebSocket = (url: string) => {
  const [socket, setSocket] = useState<WebSocket | null>(null)

  const connect = () => {
    const ws = new WebSocket(url)

    ws.onmessage = (event) => {
      const data = JSON.parse(event.data)

      switch (data.type) {
        case 'transcription':
          // 자막 + 음성 처리
          break
        case 'overlay':
          // 오버레이 처리
          break
      }
    }

    setSocket(ws)
  }

  const send = (data: object) => {
    socket?.send(JSON.stringify(data))
  }

  return { connect, send }
}
```
