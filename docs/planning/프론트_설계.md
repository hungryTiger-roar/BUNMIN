# 프론트엔드 설계

## 개요

React + TypeScript 기반 웹앱.
Electron 데스크탑 앱으로도 패키징 가능하며, 강의자와 수강자 모두 동일한 앱을 사용하고 역할만 다릅니다.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Framework | React + TypeScript |
| 번들러 | Vite |
| 데스크탑 (선택) | Electron (루트 package.json에서 관리) |
| 상태 관리 | Zustand |
| 스타일링 | Tailwind CSS |
| 실시간 통신 | WebSocket (native) |

---

## 화면 구성

### 1. 홈 (역할 선택)
- 강의자로 시작 / 수강자로 참여 선택

### 2. 로딩
- 백엔드 AI 모델 로딩 진행률 표시
- 모델별 로딩 상태 (ASR / NMT / TTS / OCR)

### 3. 강의자 화면
- **강의자료 모드**: PDF 업로드 → 슬라이드 뷰어로 페이지 제어
- **화면공유 모드**: 화면 캡처 후 수강자에게 실시간 전송
- 마이크 ON/OFF → 음성 실시간 번역
- 강의 시작 / 일시정지 / 종료 제어
- 실시간 자막 미리보기

### 4. 수강자 화면
- 슬라이드 또는 화면공유 수신 표시
- 번역 음성 재생 ON/OFF
- 자막 표시 ON/OFF
- 원본/번역 토글
- 원본 PDF 다운로드

```

│   │   ├── common/
│   │   │   ├── SubtitleDisplay.tsx   # 자막 표시

│   ├── hooks/
│   │   ├── useWebSocket.ts       # WebSocket 연결 + 메시지 처리
│   │   ├── useAudioCapture.ts    # 마이크 캡처 (500ms 청크)
│   │   └── useScreenCapture.ts   # 화면 캡처 (JPEG, 2fps)
│   │
│   ├── stores/
│   │   └── lectureStore.ts       # 전역 상태 (Zustand)
│   │
│   └── lib/

                                  # Electron 스크립트/deps는 루트 package.json에서 관리
```

## Electron ↔ 백엔드 통신

```
Electron main.cjs
  ├── spawn aunion_backend.exe
  ├── stdout 파싱 → __AUNION_STATUS__ JSON으로 로딩 진행률 수신
  ├── health 폴링 (backup, port 18765)
  └── IPC: get-lan-ip → LAN IP 반환
```

## 상태 관리 (Zustand - lectureStore)

| 상태 | 타입 | 설명 |
|------|------|------|
| isConnected | boolean | WebSocket 연결 여부 |
| isLectureStarted | boolean | 강의 진행 중 |
| isPaused | boolean | 일시정지 상태 |
| presentationMode | 'slide' \| 'screen' | 발표 모드 |
| isMicOn | boolean | 마이크 상태 |
| slideId | string \| null | 현재 슬라이드 ID |
| slideStatus | 'none' \| 'uploading' \| 'processing' \| 'ready' | 슬라이드 상태 |
| currentPage | number | 현재 페이지 번호 |
| totalPages | number | 전체 페이지 수 |
| slidePages | SlidePage[] | 슬라이드 이미지 URL 목록 |
| subtitles | Subtitle[] | 실시간 자막 (최근 50개) |
| isAudioOn | boolean | 번역 음성 재생 여부 |
| isSubtitleOn | boolean | 자막 표시 여부 |

## URL 처리 (Electron / 웹 자동 감지)

`src/lib/api.ts`에서 환경을 자동 감지합니다:

```typescript
// Electron: localhost:8000 (내장 백엔드)
// 웹: 현재 origin 기준
export const API_BASE = isElectron ? 'http://localhost:8000' : ''
export const WS_BASE  = isElectron ? 'ws://localhost:8000' : 'ws://...'
```
