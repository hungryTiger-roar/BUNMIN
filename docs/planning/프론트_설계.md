# 프론트엔드 설계

## 개요

React + TypeScript 기반 웹앱.
Electron 데스크탑 앱으로도 패키징 가능하며, 강의자와 수강자 모두 동일한 앱을 사용하고 역할만 다릅니다.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| Framework | React + TypeScript |
| 번들러 | Vite |
| 데스크탑 (선택) | Electron (루트 package.json에서 관리) |
| 상태 관리 | Zustand |
| 스타일링 | Tailwind CSS |
| 실시간 통신 | WebSocket (native) |

---

## 화면 구성

### 1. 홈 (역할 선택)
- 강의자로 시작 / 수강자로 참여 선택

### 2. 참여 (수강자 이름 입력)
- 이름 입력 후 강의 참여
- 이름 로컬스토리지 저장 옵션

### 3. 로딩
- 백엔드 AI 모델 로딩 진행률 표시
- 모델별 로딩 상태 (ASR / NMT / TTS / OCR)

### 4. 강의자 화면
- **강의자료 모드**: PDF 업로드 → 슬라이드 뷰어로 페이지 제어
- **화면공유 모드**: 화면 캡처 후 수강자에게 실시간 전송
- 마이크 ON/OFF → 음성 실시간 번역
- 강의 시작 / 일시정지 / 종료 제어
- 실시간 자막 미리보기

### 5. 수강자 화면
- 슬라이드 또는 화면공유 수신 표시
- 번역 음성 재생 ON/OFF
- 자막 표시 ON/OFF
- 원본/번역 토글
- 원본 PDF 다운로드

## Electron ↔ 백엔드 통신

```
Electron main.cjs
  ├── spawn aunion_backend.exe
  ├── stdout 파싱 → __AUNION_STATUS__ JSON으로 로딩 진행률 수신
  ├── health 폴링 (backup, port 18765)
  └── IPC: get-lan-ip → LAN IP 반환
```

## 상태 관리 (Zustand - lectureStore)

| 상태 | 타입 | 설명 |
|------|------|------|
| modelMode | 'idle' \| 'slide' \| 'switching' \| 'realtime' | AI 모델 현재 로드 모드 |
| isConnected | boolean | WebSocket 연결 여부 |
| isLectureStarted | boolean | 강의 진행 중 |
| isPaused | boolean | 일시정지 상태 |
| presentationMode | 'slide' \| 'screen' | 발표 모드 |
| isMicOn | boolean | 마이크 상태 |
| currentScreen | string \| null | 화면공유 캡처 데이터 (base64) |
| slideId | string \| null | 현재 슬라이드 ID |
| slideStatus | 'none' \| 'uploading' \| 'processing' \| 'ready' | 슬라이드 처리 상태 |
| currentPage | number | 현재 페이지 번호 |
| totalPages | number | 전체 페이지 수 |
| slidePages | SlidePage[] | 슬라이드 이미지 URL 목록 |
| studentName | string | 수강자 이름 (Start.tsx에서 입력) |
| studentCount | number | 현재 접속 중인 수강자 수 |
| viewMode | 'original' \| 'translated' | 수강자 원본/번역 토글 |
| isAudioOn | boolean | 번역 음성 재생 여부 |
| isSubtitleOn | boolean | 자막 표시 여부 |
| subtitleSettings | \{ fontSize, position, opacity \} | 자막 커스터마이징 설정 |
| subtitles | Subtitle[] | 실시간 자막 (최근 50개) |

## URL 처리 (Electron / 웹 자동 감지)

`src/lib/api.ts`에서 환경을 자동 감지합니다:

```typescript
// Electron: localhost:8000 (내장 백엔드)
// 웹 개발: '' (빈 문자열) — Vite 프록시가 /api, /slides, /ws 등을 백엔드로 중계
export const API_BASE = isElectron ? 'http://localhost:8000' : ''

export const WS_BASE = isElectron
  ? 'ws://localhost:8000'
  : `${window.location.protocol === 'https:' ? 'wss:' : 'ws:'}//${window.location.host}`

export const WS_PIPELINE_URL = `${WS_BASE}/ws/pipeline`
```

웹 개발 환경에서 백엔드 주소는 `BACKEND_URL` 환경변수로 제어합니다 (기본값: `http://localhost:8000`). 자세한 내용은 [환경설정](../setting/환경설정.md)을 참고하세요.
