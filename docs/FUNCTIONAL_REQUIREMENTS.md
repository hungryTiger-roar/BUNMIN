## 1. Electron 데스크톱 앱

### 1.1 앱 실행 및 초기화

| ID | 분류 | 기능명 | 요구사항 |
|----|------|--------|---------|
| E-FR-001 | 앱실행 | 프로그램 시작 | 사용자는 데스크톱 아이콘을 더블클릭하여 프로그램을 실행할 수 있어야 한다. |
| E-FR-002 | 앱실행 | Electron 앱 로드 | 시스템은 Electron 28.2.0 기반으로 앱을 실행해야 한다. |
| E-FR-003 | 앱실행 | Vite 개발 서버 연결 | 시스템은 개발 모드에서 Vite 개발 서버(localhost:3000)에 연결해야 한다. |
| E-FR-004 | 앱실행 | React 앱 로드 | 시스템은 React 18.2.0 기반 웹 앱을 Electron 창에 로드해야 한다. |
| E-FR-005 | 앱실행 | 백엔드 연결 확인 | 시스템은 백엔드 서버(localhost:8000) 연결 상태를 확인해야 한다. |

## 3. 교수님(강의자) 기능

### 3.2 마이크 제어

| L-FR-022 | 화면공유 | 전체 화면 공유 | 교수님은 전체 모니터 화면을 공유할 수 있어야 한다. |

| L-FR-028 | 화면공유 | 상태 표시 | 시스템은 화면공유 상태를 "켜짐" 또는 "꺼짐"으로 명확히 표시해야 한다. |
