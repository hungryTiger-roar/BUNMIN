할루시네이션 감소
1 세그먼트 신뢰도 필터링 (ASR) asr_service.py 쉬움
faster-whisper는 세그먼트마다 no_speech_prob(무음 확률)과 avg_logprob(평균 로그 확률)을 제공합니다. 현재 코드는 이를 무시하고 전체를 합치는데, 임계값 필터를 추가하면 잡음·무음 구간의 환각을 제거할 수 있습니다. 
# asr_service.py — transcribe() 내부
segments, _ = self.model.transcribe(audio_array, ...)

result_texts = []
for seg in segments:
    if seg.no_speech_prob > 0.6:   # 무음/잡음 구간 제거
        continue
    if seg.avg_logprob < -0.8:     # 확신 낮은 추론 제거
        continue
    result_texts.append(seg.text)

return "".join(result_texts).strip()
2 VAD 파라미터 튜닝 (ASR) asr_service.py 쉬움
VAD(Voice Activity Detection)의 무음 판단 기준을 조정해 발화 경계를 더 정확히 감지합니다. 
segments, _ = self.model.transcribe(
    audio_array,
    vad_filter=True,
    vad_parameters=dict(
        min_silence_duration_ms=300,  # 무음 판단 기준 (ms)
        speech_pad_ms=200,            # 발화 앞뒤 여유 구간
    ),
)
3 반복 억제 패널티 (MT) nmt_service.py 쉬움
번역 모델이 같은 단어를 반복하는 루프 환각에 대응합니다. repetition_penalty와 no_repeat_ngram_size로 억제할 수 있습니다. 
generate_kwargs = dict(
    **inputs,
    max_length=max_length,
    num_beams=1,
    repetition_penalty=1.3,    # 반복 단어 생성 억제
    no_repeat_ngram_size=3,     # 3-gram 반복 차단
)

속도 향상
4 beam_size 축소 (ASR) asr_service.py 쉬움
기본값 beam_size=5를 1(greedy decoding)로 줄이면 품질 손실은 거의 없으면서 ASR 속도가 약 30% 향상됩니다. 
segments, _ = self.model.transcribe(
    audio_array,
    beam_size=1,  # greedy — 품질 거의 동일, ~30% 속도 향상
    ...
)
5 동적 max_new_tokens (MT) nmt_service.py 쉬움
max_length=512는 입력+출력 합계를 제한해 긴 입력일수록 출력 예산이 줄어드는 문제가 있습니다. max_new_tokens로 교체하고 입력 토큰 수 기반으로 동적 산정하면 짧은 문장에서 불필요한 탐색이 사라집니다. 
# 입력 토큰 수 기반으로 출력 예산 동적 산정
# 한국어→영어는 토큰 수가 보통 1.5~2배로 증가
input_token_len = inputs["input_ids"].shape[1]
max_new_tokens = max(30, input_token_len * 2)

generate_kwargs = dict(
    **inputs,
    max_new_tokens=max_new_tokens,
    num_beams=1,
)
6 NMT + TTS 결과 캐시 (파이프라인) ws.py 중간
강의에서 반복되는 표현("예를 들어", "다음 슬라이드")은 MT+TTS를 건너뛰고 캐시에서 즉시 응답합니다. LRU 방식으로 최신 100개를 유지합니다. 
from collections import OrderedDict

_cache: OrderedDict = OrderedDict()
CACHE_MAX = 100

def cache_get(key):
    if key in _cache:
        _cache.move_to_end(key)
        return _cache[key]
    return None

def cache_put(key, value):
    _cache[key] = value
    _cache.move_to_end(key)
    if len(_cache) > CACHE_MAX:
        _cache.popitem(last=False)

# process_audio() 내부 — ASR 이후
cached = cache_get(korean_text)
if cached:
    english_text, audio_bytes = cached
else:
    english_text = await asyncio.to_thread(_nmt_service.translate, ...)
    audio_bytes  = await asyncio.to_thread(_tts_service.synthesize, ...)
    cache_put(korean_text, (english_text, audio_bytes))
7 CTranslate2로 MT 변환 nmt_service.py 어려움
Helsinki opus-mt 모델을 CTranslate2 포맷으로 변환하면 CPU 추론 속도가 3~5배 향상됩니다. 모델 변환과 라이브러리 교체가 필요합니다.
